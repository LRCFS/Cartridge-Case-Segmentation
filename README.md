# Cartridge-case-segmentation
NIST Cartridge case segmentation using UNet, Dense UNet and Deeplabv3+ 

# Pretrained Models
You can download [here](https://drive.google.com/drive/folders/1EyY8MzM3SpOniKnLYAbc9dTJpu1wIDCU?usp=sharing)

# Test your own image
If you want to test your own image, place your image and HDF5 files in the relevant directory and adjust the patch_size and stride according to your image size.
